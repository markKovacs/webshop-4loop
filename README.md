# 4loop Webshop
Basic webshop to practice Spark, Thymeleaf multi-layer architecture and JDBC.
Created by 4loop team: Krisztián Alt, Péter Juhász, Márk Kovács.

## Features
- Account Login / Registration
- Edit Profile (set default address info for later orders)
- Add to Cart (new items, increase quantity)
- Review Page (manage quantities or remove items)
- Checkout Page (add address info manually or use default loaded from profile)
- Payment Page (select payment method, both are dummy implementations right now)
- Order History (all orders saved to database and can be queried)

## Config
Rename ConfigTemplate class file to Config and fill in all the fields.
